# **App Name**: LetterBookFront Revamp

## Core Features:

- Complete TypeScript Refactor: Convert remaining .jsx files to .tsx, ensuring comprehensive data typing across the application.
- Component Optimization: Refactor page logic from src/pages directly into src/app, utilizing Server Components where applicable to improve performance.
- UX Enhancement with Loading Feedback: Implement visual loading indicators (e.g., Skeletons) and improve error handling in API calls for a smoother user experience.
- Revamp with Next.js 15 App Router: Leverage server-side rendering (SSR) to make a SEO-friendly book catalog.

## Style Guidelines:

- Primary color: Highlight green (#00E054), drawing inspiration from Letterboxd.
- Background color: Dark gray (#121212), which serves to establish a modern dark mode aesthetic.
- Accent color: A lighter shade of green, approximately (#33FF66), to provide sufficient contrast to the primary while maintaining harmony.
- Body and headline font: 'Inter', a grotesque-style sans-serif offering a modern, neutral aesthetic suitable for both headings and body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Lucide React icon library. Maintain consistency in the usage of icons.
- Ensure the design leverages Tailwind CSS and Material UI to create a fully responsive layout across all devices.
